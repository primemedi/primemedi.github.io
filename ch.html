IRF

基于 RF 的 ICE 冷却身体瘦身系统

RF System

一般脂肪在41°C就会开始溶解。
比起非肥胖者，肥胖者的脂肪组织结扎更为扎实，而且在坚硬的脂肪组织的之间有不必要的水分。
高频所产生的深热使坚硬的脂肪组织结扎松弛，以产生溶解和燃烧水分的作用，并加快脂肪组织燃烧。 脂肪组织被溶解和燃烧后，变成汗水和小便排除来。

Principle of ICE Therapy

ICE 肥胖疗法:Maintaining homeostasis 体温下降时，人体进入紧急状态，以保持温度。心脏以更快的速度输送血液， 从而整个人体的血液循环变得更快。而且，大量新鲜血液往冷却的部位移动，从而产生自然治愈的作用，这个过程中， 可使积累于体内的脂肪分解。

Cooling Point

据说，从人体的正常体温36.5°C下降8°C时，脂肪燃烧代谢最为活跃。

本产品具有强力的Ice Cooling效果，最多下降20°C，因此用身体接触手把时，容易下降8°C.

Cooling Relaxation

它搞活用-20°C的超强Cooling冷却的皮肤表面细胞，以提高渗透的药物的吸收率。
减少施术后Downtime和PIH的危险性。

Penetration of Solution

CRYO-ELECTROPORATION（冷冻电气穿孔法）是指，用电脉冲扰乱构成细胞膜的卵磷脂双重膜，在皮肤表面上产生临时性的小洞， 让有效药物（40，000 Dalton以下）渗透到细胞内部

VRF

基于 RF 的真空身体瘦身系统

RF System

Multi-polar RF 的极性变化而在相当于两个电极之间的距离（长度）一般的长度（深度）产生热点， 将强力的 RF 能量传递到皮肤 dermis 层深处，让热能集中在 dermis 内的弹性纤维或弹性蛋白纤维上。 这时发生的热使 dermis 内衰老的胶原蛋白收缩，以提高弹性和改善皱纹及细纹。

Vacuum System

本器具使用真空负压促进被脂肪细胞过度压住的血液和淋巴循环，让皮肤变得健康，通过强力的吸出和震动 直接对脂肪团加压，以分解脂肪团。与此同时，它可以产生弹性效果，让身体变得苗条且健康。

RF System Effect

1.提高组织功能
2.增进细胞功能
3.促进淋巴及血液学环
4.去除活性氧(衰老)
5.增进新陈代谢
6.改善皮肤皱纹和提高弹性
7.缓解炎症和杀菌效果（护理青春痘）

Vacuum System Effect

1.护理脂肪团
2.对血管及淋巴管闭塞现象的治疗
3.松弛肌肉及恢复疲劳
